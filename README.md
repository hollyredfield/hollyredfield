### 🎓 Mi Experiencia como Desarrollador Junior

Como recién graduado en **Desarrollo de Aplicaciones Multiplataforma** (promoción 2025), mi formación académica se ha centrado principalmente en:

- **Java Avanzado**: El lenguaje protagonista de mis dos años de DAM, desde programación orientada a objetos hasta Spring Boot
- **Python**: Nuestro aliado para scripting, automatización, análisis de datos y desarrollo web con Django/Flask
- **Bases de Datos Relacionales**: PostgreSQL y MySQL, diseño de esquemas, consultas complejas y optimización
- **Desarrollo Móvil**: Android Studio para aplicaciones nativas Android
- **Desarrollo Web**: HTML5, CSS3, JavaScript y frameworks modernos como React
- **Sistemas**: Linux, administración de sistemas y scripting en Bash

### 🔍 Aplicación Real en NXERIO

La experiencia en NXERIO me ha permitido trasladar los conocimientos teóricos del grado a un entorno profesional real:

**🎯 Del Aula a la Empresa:**
- **Web Scraping**: De ejercicios académicos con bibliotecas básicas a sistemas robustos que procesan miles de productos diarios
- **APIs REST**: Integración real con sistemas externos, manejo de autenticación OAuth y gestión de tokens
- **Optimización**: De teoría sobre complejidad algorítmica a optimización real para entornos de producción
- **WordPress**: Desarrollo de plugins personalizados (algo completamente nuevo fuera del temario académico)
- **Testing de Rendimiento**: Pruebas reales con cargas de trabajo de más de 10.000 registros

**🚀 Tecnologías Nuevas Aprendidas:**
- **React**: Framework no visto en profundidad en DAM, aprendido durante las prácticas
- **Capacitor**: Para desarrollo híbrido multiplataforma
- **Supabase**: Como alternativa moderna a bases de datos tradicionales
- **Vite**: Herramientas de build modernas para desarrollo frontend
- **Tailwind CSS**: Framework CSS utilitario para diseño rápido

### 🧠 Lecciones Aprendidas

**Diferencias entre Desarrollo Académico y Profesional:**
- En DAM: Proyectos individuales con especificaciones claras
- En NXERIO: Colaboración en equipo, requisitos cambiantes, sistemas legacy

**Competencias Técnicas vs. Competencias Blandas:**
- Comunicación con stakeholders no técnicos
- Documentación para futuros desarrolladores
- Gestión de tiempo en entornos con deadlines reales
- Debugging en sistemas complejos en producción

### 🎯 Próximos Objetivos Profesionales

Como desarrollador junior, mis objetivos a corto-medio plazo incluyen:
- **Arquitecturas de Microservicios**: Profundizar en sistemas distribuidos
- **Testing Avanzado**: TDD, BDD, testing de integración y e2e
- **DevOps**: Docker, Kubernetes, CI/CD pipelines
- **Especialización Backend**: APIs GraphQL, caching distribuido, message queues
- **Certificaciones**: Spring Professional, AWS Cloud Practitioner
